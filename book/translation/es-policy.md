# Spanish Translation Policy

This policy governs the Latin American Spanish edition of `Agentic Systems Patterns`.

## Editorial Rules

- Use clear Latin American Spanish for prose.
- Keep paths, slugs, filenames, anchors, and URLs unchanged.
- Keep code blocks, inline code, commands, environment variables, package names, API names, class names, function names, and configuration keys unchanged.
- Keep framework and protocol names unchanged.
- Keep the approved technical vocabulary in English.
- Translate surrounding explanations, transitions, examples, warnings, and editorial copy.
- Preserve Markdown structure, frontmatter keys, tables, admonitions, links, images, diagrams, and heading levels.
- Do not add new claims, examples, warnings, citations, or recommendations during translation.
- Prefer direct, active, concise Spanish.

## Examples

> El agent loop mantiene el state, invoca tools, registra traces y se detiene cuando el goal se completa.

English:

> Use this pattern when the workflow needs durable checkpoints and replayable traces.

> Usa este pattern cuando el workflow necesita checkpoints durables y traces reproducibles.

[ ] Source references recorded

[ ] Consent state recorded where required

Retrieval
[ ] Retrieval filters by tenant before ranking
[ ] Retrieval filters by actor and permissions
[ ] Retrieval filters by memory kind and retention
[ ] Retrieved memory is labeled as memory, not instruction
[ ] Stale memory loses to fresh evidence
[ ] Conflicting memory is surfaced or escalated

[ ] Writes are traced
[ ] Denials are traced
[ ] Corrections are traced
[ ] Deletions are traced
[ ] Expiry and index removal are tested
[ ] Write kill switch exists
[ ] Read kill switch exists

[ ] Fixture covers allowed preference write

[ ] Fixture covers review-required untrusted write
[ ] Fixture covers correction
[ ] Fixture covers deletion
[ ] Fixture covers stale memory
[ ] Fixture covers tenant isolation
[ ] Fixture covers memory not overriding instructions

[ ] Green: ready for controlled use

# Book Releases

Generated offline artifacts live here.

- `Agentic-Systems-Patterns.pdf`: printable courtesy snapshot of the current book.
- `Agentic-Systems-Patterns.epub`: reflowable courtesy snapshot of the current book.

The GitHub Pages site is the primary reading experience. PDF and EPUB files are checked into the repository as courtesy formats for offline reading. Their book/reference content is licensed under Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International (`CC-BY-NC-SA-4.0`). Code examples remain licensed under MIT. If they grow too large, move future courtesy formats to GitHub Releases and keep this file as the pointer.

Generate them locally with:

```bash
npm run book:pdf
npm run book:epub
```

The deployment workflow also generates the courtesy formats and publishes them at:

```text
https://gturitto.github.io/Agentic-Systems-Patterns/releases/Agentic-Systems-Patterns.pdf
https://gturitto.github.io/Agentic-Systems-Patterns/releases/Agentic-Systems-Patterns.epub
```

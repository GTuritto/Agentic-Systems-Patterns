# AutoGen Transcript Pattern

This folder supports Lab 13. It is a deterministic AutoGen-style reference implementation that focuses on the architectural contract: named agents, team turns, structured messages, transcript ownership, termination, and transcript evals.

Run the demo:

```bash
npm run autogen-transcript
```

Run the tests:

```bash
npm run autogen-transcript:test
```

The code intentionally avoids live model calls so the lab can run without API keys.
